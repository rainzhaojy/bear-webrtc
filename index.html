WebRTC Examples

如果两个页面需要signaling server来交换signaling message，则需要运行"node server.js"启动websocket server，这个wss会广播收到的消息，相当于是一个signaling server，这样两个页面可以实现信令交换.

WebRTC basic (signaling server required):

  1. basic.html - 一个最简单的webrtc demo，可以打开两个basic.html相互通讯
  2. audio.html - audio only webrtc demo, 可以打开两个audio.html相互通讯
  3. steps.html - 需要自己串起每一步, 可以一端打开basic.html, 另一端打开steps.html
  4. roap.html - 使用roap.js实现signaling，可以打开两个roap.html相互通讯
  5. audiofromfile.html - audio stream from file, 可以一边打开audio.html，一边打开audiofromfile.html相互通讯

getUserMedia (signaling server not required):

  1. resolution.html - 利用getUserMedia()第一个参数constraints限定video resolution, 切换resolution之前会clean up现有video
  2. mediasource.html - 利用MediaStreamTrack.getSources()得到audio/video输入源，然后利用getUserMedia()的constraints参数限定source
  3. screenshot.html - 利用getUserMedia()与canvas实现拍照功能
  4. volume.html - 利用getUserMedia()得到audio stream, 用Web Audio API得到audio volume

Web Audio API

  1. webaudio.html - 一个简单的Web Audio API demo，使用XHR下载mp3文件并使用WEB AUDIO API播放
  2. volume.html - 利用getUserMedia()得到audio stream, 用Web Audio API得到audio volume
  3. audiofromfile.html - audio stream from file, 可以一边打开audio.html，一边打开audiofromfile.html相互通讯

Others:

  1. media from file - 不用getUserMedia()而是从文件中得到media stream
